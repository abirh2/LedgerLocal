## [0.0.0] - 2026-07-14

### Added
- Initial local-first LedgerLocal release.
- Statically hosted React 19 single-page application built on Vite 6.
- In-browser local storage persistency leveraging IndexedDB.
- Columns mapping wizard for customized financial CSV imports.
- Top-down rule evaluation engine supporting nested conditions and priority execution.
- Multi-profile database isolation.
- Interactive spending dashboards and charts designed using Recharts.
- Offline budgeting tool with rollover features.
- Recurring transaction scanner and expected payment logs.
- Fictional data seed engine for immediate system exploration.
